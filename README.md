# Paralism
Paralism is an new economy based on Parallel Blockchain Technology.

## Prerequisite 
### Windows: 
    Microsoft Visual Studio 2015
    CppRestSDK 2.9.1 and above
    sqlite 3 and above
### Linux:
    CppRestSDK 2.9.1 and above
    sqlite 3 and above
